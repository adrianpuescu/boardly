# Security

If you discover a vulnerability in this project (including the deployed app at [boardly.webz.ro](https://boardly.webz.ro)), please report it responsibly.

**Please do not** open a public GitHub issue for undisclosed security bugs.

Instead, contact the maintainer via a **private channel** (e.g. GitHub Security Advisories for this repository, or email if listed on the maintainer’s GitHub profile).

Include enough detail to reproduce: affected route or component, steps, and impact. You should receive an acknowledgment within a few business days.

---

This is an open-source portfolio project. There is no formal bug bounty program.
